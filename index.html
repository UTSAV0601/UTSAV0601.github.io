Projects

FLAMEBOY

Project description

A 2D adventure game where you dodge hazards falling randomly from sky and player health decreases as it collides with hazards.

Game Engine: Unity
Implementation: C# in Visual Studio 2019

This project has been my major breaktrough as I learnt many things a game consist of while making it. It was my first project and my first tryst with game making. Below are the things I learnt from this project.

  • Character creation using Photoshop and Gimp
  • Character Movement Hazard random spawn and integrating corresponding animations
  • Special effect such as particle effectand Implementation of box collider
  • Random speed of the hazards and player HUD
  • Dash ability to the player
  • Sound effects
  • Start and Pause Menu UI
